# 🛒 Amazon Clone (HTML & CSS)

## 📌 Description

This is a front-end clone of the Amazon website built using HTML and CSS.
The project focuses on replicating the layout, design, and user interface of Amazon's homepage.

## 🚀 Features

* Responsive navigation bar
* Product sections and cards
* Banner/hero section
* Footer similar to Amazon
* Clean and structured layout



## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Folder Structure

```
Amazon-Clone/
│── index.html
│── style.css
│── images/
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/naifsheikh0808-collab/amazon-clone.git
```

2. Open `index.html` in your browser
